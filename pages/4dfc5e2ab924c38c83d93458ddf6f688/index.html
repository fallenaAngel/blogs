# 使用attr()抓取自定义属性值

<html>
  <div class="bruce flex-ct-y">
    <a class="tips" href="https://www.baidu.com" data-msg="Hello World">提示框</a>
    <a class="tips" href="https://www.baidu.com"></a>
  </div>
</html>
<style type="text/css" scoped>
  .flex-ct-y {
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
  }
  .tips {
    position: relative;
    display: block;
    margin-top: 10px;
    padding: 0 20px;
    border-radius: 10px;
    height: 40px;
    background-color: #66f;
    line-height: 40px;
    color: #fff;
    width: auto;
  }
  .tips::after {
    position: absolute;
    left: 0;
    top: 0;
    border-radius: 5px;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.5);
    opacity: 0;
    text-align: center;
    font-size: 12px;
    content: attr(data-msg);
    transition: all 300ms;
  }
  .tips:first-child {
    margin-top: 0;
  }
  .tips:hover::after {
    left: calc(100% + 20px);
    opacity: 1;
  }
  .tips[href]:empty::before {
    content: attr(href);
  }
  /* empty表示：没有子元素（包括子标签、文本、空格）的标签 */
  .tips[href]:empty:hover::after {
    display: none;
  }

</style>

# 使用linear-gradient控制背景和文本渐变

<html>
  <h3>背景渐变</h3>
  <p>
    要点：通过linear-gradient设置背景渐变色并放大背景尺寸，添加背景移动效果<br>
    场景：主题化、彩虹背景墙
  </p>
  <div class="bruce">
    <div class="gradient-bg">iCSS</div>
  </div>
  <h3>文本渐变</h3>
  <p>
    要点：通过linear-gradient设置背景渐变色，配合background-clip:text对背景进行文本裁剪，添加滤镜动画<br>
    场景：主题化、特色标题
  </p>
  <div class="bruce flex-ct-x">
    <h1 class="gradient-text">Full Stack Developer</h1>
  </div>
</html>
<style type="text/css" scoped>
  .gradient-bg {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100%;
    background: linear-gradient(135deg, #f66, #f90, #3c9, #09f, #66f) left center/400% 400%;
    font-weight: bold;
    font-size: 100px;
    color: #fff;
    -webkit-animation: move 10s infinite;
            animation: move 10s infinite;
  }

  @-webkit-keyframes move {
    0%,
    100% {
      background-position-x: left;
    }
    50% {
      background-position-x: right;
    }
  }

  @keyframes move {
    0%,
    100% {
      background-position-x: left;
    }
    50% {
      background-position-x: right;
    }
  }
  .gradient-text {
    background-image: linear-gradient(90deg, #f66, #f90);
    -webkit-background-clip: text;
            background-clip: text;
    line-height: 60px;
    font-size: 60px;
    -webkit-animation: hue 5s linear infinite;
            animation: hue 5s linear infinite;
    -webkit-text-fill-color: transparent;
  }

  @-webkit-keyframes hue {
    from {
      -webkit-filter: hue-rotate(0);
              filter: hue-rotate(0);
    }
    to {
      -webkit-filter: hue-rotate(-1turn);
              filter: hue-rotate(-1turn);
    }
  }

  @keyframes hue {
    from {
      -webkit-filter: hue-rotate(0);
              filter: hue-rotate(0);
    }
    to {
      -webkit-filter: hue-rotate(-1turn);
              filter: hue-rotate(-1turn);
    }
  }
</style>
