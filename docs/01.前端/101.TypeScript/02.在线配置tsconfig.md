## 网站

Ts Config Helper 是一个功能强大、易用性高的 TypeScript 配置工具，提供可视化配置、详细Ts官网文档查阅、预设配置列表和分类过滤等多种功能，帮助你更快速、准确地完成 tsconfig.json 的配置。

* 可视化配置
* 文档查阅
* 预设配置
* 分类过滤
* 双向解析

[地址](https://tsconfiger.netlify.app/)
